Difference b/w document and window objects

Document :
1. It represent HTML document
2. It is object of window property
3. Tags, elements & attributes are part of the document
4. Syntax : Document.property name
5. Document.title will return the title of th e document

Window Objects
1. It represent browser window
2. It is the object of browser
3. Variables, object & functions are part of the browser
4. Syntax : Window.property name
5. Window.innerheight will return the height of the content area.
